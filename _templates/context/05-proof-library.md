# Proof Library

> **How to use this file:** Document every proof point approved for use in AI-generated outputs. Organize by outcome type, persona, and objection so AI can select the most resonant evidence for a given situation — not just the most impressive stat. Every entry must be cleared for external use before being added. Upload to your Claude Project.

---

## Proof Point Standards

Before adding any proof point, confirm:
- [ ] Approved by legal / marketing for external use
- [ ] Customer has consented (named reference) OR anonymized appropriately
- [ ] Metric is accurately represented (not cherry-picked timeframe, correct denominator)
- [ ] Context is documented (pilot vs. production, customer size, how long they'd been a customer)
